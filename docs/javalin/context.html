Context

Methods can use io.javalin.Context as a method argument. The web route will pass the context on to the controller method.

We want to do this when we want full access including things like headers, cookies etc. 

@Post
void save(HelloDto dto, Context context) {
  // use javalin context, get cookies, headers ...
}
