Documentation / Gradle

Gradle

See the example at: examples/javalin-gradle-java-basic/build.gradle

Add dinject as a dependency. 

dependencies {
    ...
    compile('io.dinject:dinject:1.10')
    ...
}

APT - dinject-generator

Add dinject-generator as a annotation processor. 

dependencies {
    ...
    annotationProcessor('io.dinject:dinject-generator:1.10')
    ...
}

Kotlin KAPT - dinject-generator

See example at: https://github.com/dinject/examples/blob/master/basic-di-kotlin-maven/pom.xml

For use with Kotlin we register dinject-generator as a KAPT processor to the Kotlin compiler. 

dependencies {
    ...
    kapt('io.dinject:dinject-generator:1.10')
    ...
}

The above includes the dinject-generator annotation processor only. For use with Javalin we additionally include the io.dinject:javalin-generator:1.9 tile.
